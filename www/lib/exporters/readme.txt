Библиотека классов для экспортирования данных в файлы.

Экспортирование происходит в директорию export. 

Для формирования пути экспортирования используется функция createPath из 
абстрактного класса.

Чтобы добавить новый формат файла для экспорта, создаем новый документ
и вставляем следующий код:

	require_once "abstract_class.php";

	class Exporter<формат_файла> extends Exporter {

		public static function export($data, $filename) {
			self::$format = "<формат файла>";
			//...
		}
	}
	
Далее остается реализовать функцию export в новом классе.
